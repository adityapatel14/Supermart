# Sales KPI & Performance Analytics Dashboard

## Overview
This project focuses on analyzing sales performance data to derive actionable business insights.  
The objective is to build KPI-driven dashboards that help stakeholders understand revenue trends, customer behavior, and overall business performance.

## Problem Statement
Businesses generate large volumes of sales data, but raw data alone does not provide insights.  
The goal of this project is to:
- Identify key sales trends
- Track performance across products and regions
- Support data-driven business decisions through dashboards

## Dataset
The dataset contains transactional sales data including:
- Order details
- Product information
- Customer segments
- Sales and profit metrics  

(Source: Sample retail sales dataset)

## Tools & Technologies
- SQL (MySQL)
- Power BI
- Microsoft Excel
- GitHub

## Key Analysis Performed
- Wrote 15+ SQL queries using:
  - `JOIN`
  - `GROUP BY`
  - Subqueries
- Calculated business KPIs such as:
  - Total Revenue
  - Total Profit
  - Average Order Value
  - Monthly Sales Growth
- Performed trend analysis across:
  - Time (monthly/quarterly)
  - Product categories
  - Regions and segments

- Revenue and profit trends
